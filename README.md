# gwt-proyecto-QA


Proyecto creado para el control y monitoreo de un equipo de QA en el cual se utilizaron las siguiente metodologias y herramientas:

*  Java v 19
*  IDE Eclipse 2023
*  GWT tecnologia de google para el desarrollo de aplicaciones robustas 
*  Principios solid
